#### A plugin for the minecraft server *hglabor.de* that adds different challenges:
- **Damagers** (Easy, Medium, Hard, Impossible, ...)
- **Mlgs** (Water, Block (Cobweb, Slime, ...), Horse, Strider, Boat, Minecart, Potion)
